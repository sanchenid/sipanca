Skema Rangkaian Kami

Gambar di atas menunjukkan skema rangkaian sistem monitoring cuaca kami. Sistem ini dirancang untuk mengintegrasikan berbagai sensor dengan ESP32, menampilkan data secara real-time, dan menyediakan solusi yang dapat disesuaikan.

Cara Penggunaan

  1. Pasang semua sensor sesuai skema rangkaian yang disediakan.
  2. Hubungkan perangkat ke sumber daya listrik.
  3. Konfigurasikan koneksi Wi-Fi melalui aplikasi yang disediakan.
  4. Buka antarmuka web atau aplikasi untuk memantau data cuaca secara real-time.
  5. Untuk troubleshooting, silakan lihat panduan teknis di dokumentasi.

Spesifikasi Alat

  • Mikrokontroler: ESP32 dengan konektivitas Wi-Fi dan Bluetooth
  • Sensor: DHT22 (Suhu & Kelembapan), BMP280 (Tekanan Udara), BH1750 (Intensitas Cahaya), Sensor Hujan, MQ135 (Gas)
  • Display: LCD ST7920
  • Power Supply/adaptor: 5V/2A
  • Dimensi: 15x10x5 cm
  • Kompatibilitas: Dapat diakses melalui aplikasi web dan mobile
